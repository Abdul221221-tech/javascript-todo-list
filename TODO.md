# Todo List App

A simple and responsive Todo List application built using HTML, CSS, and JavaScript.

## Features

- Add new tasks
- Prevent duplicate tasks
- Delete tasks
- Store tasks in Local Storage
- Data persists after page refresh
- Responsive and clean UI
- Event Delegation for efficient delete handling

## Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
- Local Storage API

## How It Works

1. Enter a task in the input field.
2. Click the "Add Todo" button.
3. The task is displayed in the list and saved to Local Storage.
4. Duplicate tasks are not allowed.
5. Click the "Delete" button to remove a task from both the UI and Local Storage.
6. Tasks remain available even after refreshing the page.
